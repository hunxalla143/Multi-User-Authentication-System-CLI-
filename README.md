# Multi-User-Authentication-System-CLI

## 1 Description
This is a Python-based CLI authentication system that allows multiple users to register, login, and manage their accounts using JSON storage.

##  Features
- User Registration
- User Login
- Reset Password
- Password Hashing (SHA-256)
- JSON File Storage
- Input Validation
- Duplicate Account Prevention
- Account Deletion

##  Security
- Passwords are hashed using SHA-256
- Plain text passwords are NOT stored
- Basic validation applied

##  Files
- main.py
- users.json

##  How to Run

1. Install Python
2. Download project files
3. Run:

```bash
python main.py
